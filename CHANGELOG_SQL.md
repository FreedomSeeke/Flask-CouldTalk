# Wechat_Flask_iPhone 数据库版本说明

## 📱 版本一：手机端与PC端均已适配
**文件名**: `Flask_WeChat_iPhone.sql`
**导出环境**: Windows 10 (MySQL 8.0.32)
**导出日期**: 2026-04-12 15:54

### 1. 版本特性
- **数据状态**: 包含完整的模拟测试数据。
- **用户数据**: 包含 ID 为 1-6 的多个用户（admin, 测试一等），且包含已加密的密码和头像路径。
- **聊天记录**: `messages` 表中保留了 35 条模拟聊天内容（包含文本、图片、音频测试消息）。
- **审计日志**: `user_activities` 表非常活跃，记录了 176 条详细的操作行为（登录、修改密码、上传文件等）。

### 2. 使用场景
- 用于本地 iPhone UI 的样式调试。
- 需要查看现成聊天效果的演示场景。

## ☁️ 版本二：仅适配PC端
**文件名**: `wechat_chat.sql`
**导出环境**: Ubuntu 22.04 (MySQL 8.0.45)
**导出日期**: 2026/03/29

### 1. 版本特性
- **数据状态**: 仅保留表结构定义，清空了业务数据。
- **业务表状态**: `messages`、`friends`、`friend_requests` 表均为空。
- **审计日志**: 仅保留极少量初始化日志。

### 2. 使用场景
- 生产环境首次部署。
- 需要一个“干净”的数据库环境进行压力测试或全新数据录入。

---

## ⚠️ 迁移注意事项
- **编码**: 两个版本均采用 `utf8mb4` 编码，支持表情符号。
- **外键约束**: 在往服务器导入 iPhone 版数据前，建议检查 `users` 表的自增 ID 是否冲突。
